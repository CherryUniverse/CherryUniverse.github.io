Celestial Anarchy Archive's

The Anarchy Archive is a service tagged specifically for players of the Celestial Anarchy Terraria Server, Featuring a HTTP directory browsing web server users are able to browse a super simple web interface to download the previous world files for the Celestial Anarchy Terraria Server.

 

Anarchy Archive is also the first Cherry Service to use the Cherry Networking Platform. Featuring a None SSL server environment that updates when new files are added!
