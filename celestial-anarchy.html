Welcome to the Anarchy

 

Not your typical Anarchy Server, Meet Celestial Anarchy. Part of the Celestial Game Server brand we offer a none hacker friendly Anarchy server where there are no rules but no hacking.

 

Offering a different flavour of Anarchy Game-play style Celestial Anarchy is a new server with new prospects that makes sure community and freedom is on the table without the risk of hackers.

 

Celestial Anarchy also provides a new style of Rolling Reset, Featuring Bi-Monthly resets the server keeps on-top of dead accounts and remains fun throughout the days while providing a world archive where players can download the past and previous server worlds of the Anarchy.

 

All for free, and all in the name of fun!

Server Rules!

These are the rules of the server that all users must ad-hear to, Violating these server rules will earn you 1 warning followed by the banning of your account.

 

 

  • Hacking is strictly prohibited.
  • Use of hack cheats is prohibited (Stock game dupe's are allowed)
  • Causing damage to spawn or blocking the exit of spawn is prohibited and can result in a ban.
  • Causing the server to crash using lag machines or any client related server crash tools (Results in instant ban)

 

 

Rules on this server are in place to protect the player-base and keep the server fair an playable.

 

Players are allowed to use the following cheats

 

 

  • Stock game Wall glitches
  • Stock game Speed Glitches
  • Stock game Item Duplication Glitches
  • Anything that is not related to a hack client

 

 

Please note that this is not a true anarchy server, there are rules in place to protect younger children and to protect the server and players ability to play the server.

 

Thank you for adhearing to the rules.
