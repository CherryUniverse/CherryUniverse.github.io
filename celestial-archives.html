Celestial Anarchy Archive's

What is the Anarchy Archive?

 

The Celestial Anarchy Archive's is a highly requested feature for the Celestial Anarchy Server which allows for players to download the older worlds that used to run on the Anarchy Server so they can play with and explore the creations and history of the worlds that used to be!

 

Brought to users by the Rebuild Archive Project!

The Anarchy Archive

HTTP Server

The first to adopt CNP Services!

 

The Anarchy Archive is now hosted through the CNP Service using a none SSL HTTP download server. Allowing users to directory browse and download whatever Anarchy Archive world they wish to.


Despite the lack of SSL this is not required for this form of the server!
