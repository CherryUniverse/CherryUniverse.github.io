Welcome to Cherry Universe Beta!

 

Welcome to the Cherry Universe Beta, This is an early release of the upcoming Cherry Universe V20 Next Gen update and is expected to contain none web breaking bugs and glitches. Some content may also be missing or lower quality which will be ironed out by the final release.

 

You as a user of the Beta are helping us build the next generation of website but should be aware during the Beta phase you may encounter bugs that will cause problems. Report these bugs to celestial@cherryuniverse.co.uk

What’s New in Beta 7?

Welcome to Mainstream 7, This is a massive update built around and based on the feedback of people. With major optimisations to the web layout, design and language this is a massive jump for 1 beta to the next!

 

Introducing InterEX

A new technology built to allow for the advanced intergration into the CU Website for full screen games and webpps to be ran instead of Web In Web.

 

Human Design

Referring from the feedback we designed the whole interface from the ground up, With much more information, Structure and a much more human and easier to navigate interface design Beta 7 is expected to be the final major UI revision that we will be doing for V20.

 

Total Rollout

New Images, Correctly scaled icons and fixes to nearly every aspect of the website we are totally rolling out and optimising the website for the smoothest and quickest experience!

You are using a BETA website, So there are some bugs that are to be expected. We do know of some issues and faults with the website that we will list below now.

 

When using the website content many be missing or icons may have not been put in place yet. Buttons and nav items should work as expected with or without the image assets put in place.

 

Some external links may not have full functionality yet, an example of this is the Rebuild Archive where V18 and V19 are currently unable to be accessed due to being current or future builds.

 

C-NEWS also does not currently function due to being a live service, this is expected behaviour that will change when V20 takes the spotlight in 2023.

 

Some assets may appear in lesser quality than first wished, This is due to some scaling issues and is a planned fix for the future.
