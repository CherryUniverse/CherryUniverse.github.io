Celestial Flux

 

A new server in flux from the last, Celestial Flux is a Terraria server founded under the Celestial Brand. Featuring a survival builder style world game Flux aims to be a stable server providing users with a safe place to build and an enjoyable game experience.

 

With no hacking, no greifing and protected building zones Flux is a new generation of server that caters to those wishing to play on a server that has the survival element on a slow rolling reset world.

 

With a strong sense of community and powered by L-NET Celestial Flux is a unbeatable small server experience providing calm in-game community and builders paradise.

 

Celestial Flux is also cleverly tied into the rest of Cherry Universe with its elegant and powerful integration to Celestial Universe and Celestial Anarchy.
