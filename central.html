Change-log - V20.0.0

 

  • Experia Human Interface Engine, Featuring one of the most powerful additions to the CU website that we have ever seen with hundreds of new features and changes over the old Powerflow engine, From the new Adaptive Design to the Mobile Optimised UI Experia is one of the most complex web interface engines that we have ever built. Period.
  • A total Redesign down to the deepest roots of the website, Porting to a new web platform was not easy and meant we had to rebuild the entire website from the ground up, So we took the opportunity to bring you one of the biggest redesigns to the website the world has ever seen. From a new Nav Bar to a new footer, every image, every button and every icon is designed to take advantage of the new features that we have to offer even including the same redesign for the new Mobile Experia Engine and website. All of this and more in one version number update.
  • Tighter than ever web - Service integration, Through the use of the all new Inter-Site's we have tighter than ever before integration between local and web services, From Celestial Global's Anarchy and Flux websites to the new integration with the Anarchy Archive's we have overhauled the Web to Local Service integration.
  • The Gallery, Remember the classic gallery from pre V9? We have brought back the gallery better than ever before, featuring major improvements to the contents the new gallery features all the content from the old gallery bar Yuki's because she wined at me, plus a lot more content on-top of that to create the largest gallery we have ever had on Cherry Universe.
  • Cherry Central, Your Website's control centre. Replacing the legacy Updates page we have brought users a new Cherry Central where users are able to view update information, Find the experimental page and configure some website settings. All in one simple and easy to get to web page.
  • Much more graphical content, From the past Cherry Universe versions V20 is one of the most graphical updates that we have ever produced, Featuring major overhauls to old graphics in addition to many more graphical addons. V20 is a graphical overhaul with many fun and in theme graphical improvements and additions to the website.
  • The next level support system, Featuring the best level of support inside of the website that we have ever seen, From email integration to detailed instructions and information on how to get support. V20's new support centre is the most support we have ever been able to give. Ever.

Meet Celestial Sweep (V20)

 

The most powerful and largest update to Cherry Universe since its launch. With numerous changes and a total website re-write from the ground up V20 is not just ported to a new web engine, It is totally rebuild from the smallest aspects to the largest web interfaces. From PowerFlow to Experia, From Google Sites to L-NET and so much more Welcome to V20, The next generation of Cherry Universe.

 

Powered By Experia - The New Human Interface Engine

From the launch of PowerFlow with its touch based navigation UI to present day, Cherry Universe's Human interface engine has always been lacking. Today that is no more. Meet Experia, The next generation from powerflow. Featuring totally overhauled Human Interface design Experia takes the best of Powerflow, Strips away the bad and packages it in a flashy new UI, Featuring 2 tier navbar, Image By Image navigation, Inter-Sites for services that need to shine and so so much more all packed into one major overhaul and major upgrade for Cherry Universe.

 

A New Engine, A new Lease on Life

Gone are the days of restricted slow web content, Gone are the days of no work getting done when offline, Meet CU again backed by L-NET and brought to users through Adobe, With advanced and intuitive levels of control over the web design process, V20 brings a whole new UI, Design and much more to the website that you know and love with advanced HTML, CSS and JavaScript all into one powerful new Engine. And with the stability of Git-hub for hosting CU has never been so powerful.

 

Downloads, Gallery, Live Updates and much more

Not only is the UI flashy and not only is adaptive design and much more some of the most powerful and hot new features. V20 features such major new features as "The Gallery" returning for the first time since V10, Download Centre, Messaged Games and the Celestial global service. All inside one massive and advanced website in the most easy to use web format the world has ever seen.

 

V20 is a flashy new update, bringing life into every section of the website and re-engineering the website for a cleaner more fluent design. And don't forget the new Mobile Dedicated Experia engine for a mobile experience like never before. Welcome to V20!
