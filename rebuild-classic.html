Welcome to Rebuild Classic!

 

The old Rebuild UI pack, adapted from V1.4 and made to run on and with older versions of Minecraft. Retaining no new features over the older edition of the Rebuild UI pack the classic edition is the "Unofficially" Supported version of the older UI pack which retains and keeps 3D features of the old pack. (This may not work with 1.19 and newer)

 

The top features of Rebuild Classic is that it is as functional as it always has been with the older editions of Bedrock due to the LTS Support features. Rebuild Classic also features little to no changes over the version it is adapted from and retains the 1.4 version branding.

The same timeless design for old Minecraft Bedrock!
