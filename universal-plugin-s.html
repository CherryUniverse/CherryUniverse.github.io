Welcome to U-Plugin's

 

U-Plugin's, Or Universal Plug-ins are small expansion packs for Minecraft Bedrock that are designed to be paired with the Universal Suite packs.

 

U-Plugin's can be used as standalone enhancement packs but are guaranteed to be 100% functional with any of our Universal Suite packs.

 

U-Plugin's range from small texture changes such as adding Christmas chests dedicated for the Universal Winter pack, or even wind up to be more substantial changes in the way the game works.

 

All of the packs go through a verification process to make sure that they work with the latest versions of the Universal Suite. Each Pack page will have Information into supported Universal Versions.
