Support

Used with all our worlds

 

Universal Behaviour is one of the most trusted packs that we have ever had. Built to give servers which operate in a survival mode but do not want mobs the option to do so. Universal Devices moves mob spawns such as creepers and zombies to hard mode and then changes entity loot drops to ensure that the game remains playable.

 

Allowing servers to have a survival world which is peaceful but keeping the draining of huger and other resources as a thing that is happening.

 

Stable and simple like the rest of the Universal suite.

Widely functional and widely compatible

 

Universal Behaviour is widely compatible and functional with other packs, Keeping a low profile Universal Behaviour modifies the way that mobs work and spawn while keeping a low profile and widely compatible with many other packs on the market.

 

Universal Behaviour also works between different versions of Minecraft down to 1.17 to ensure that servers hosting for older versions of the game have the same capability than before.

 

Modification which is 5 star.

Download on MCPEDL

 

Get it on the central platform for Minecraft Pack distribution, Safe and secure you can get the latest version on MCPEDL

Semi Open License

 

We know how important it is for some people to need to modify and distribute packs using code from this one. So we have decided to release Universal Behaviour under our version of an open source license.

 

Users that have and download this pack are allowed to modify and even distribute copies of Universal Behaviour but must give credit to the original pack if distribution is underway.
