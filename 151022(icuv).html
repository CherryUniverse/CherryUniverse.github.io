Introducing Cherry Universe V20

Welcome to the next generation!

 

 

Woah, Things have changed around here. It looks so different and feels so strange. Welcome to the V20 update stranger.

 

Cherry Universe V20 is perhaps the single largest update that we have ever released for our website, revolutionising the way that we work and look, With a fresh new style, 4k Wide-screen support, Experia Engine, HyperEX Technology, Adaptive Engine, Newsroom, CU Download Server, CU Classic, Global and a whole slew more features and changes.

 

Today we will dive into this update and explore the good from the bad and every change that we made to our website for you. So without a further or a due, lets dive into the biggest change yet. The UI.

 

CU V20 has a new UI

 

Not noticed? Must be blind, The all new UI Includes overhauls to every section of the entire website with a fresh visual style, Human design and so much more all to improve the visual experience of our website. From a less convoluted interface with glowing and star backed design V20 brings the best of the old gen and totally re-writes it.

 

One of the largest things that have changed is the way that we did the UI, Remember all of them 3D overlapping images? Gone they are Gone, With a new glowing and star backed design style from twisting arches to twinkling icons the new UI is not only more visually pleasing but easier on the eye.

The largest thing is arguably the fact that every section of the visual design has been changed, Icons sparkle and banners glow with a simple yet elegantly crafted visual style that is not only easy to see but one of the best design eras we have ever been in.

 

The 2nd largest thing has to be the overhauled layout. Remember when pages were basically glorified text boxes? Gone. V20 introduces a brand new way to do things, Every page has click through images that work. Information is everywhere and we prioritised quick layout over form.

 

The largest change is arguably the overhauled hub. From a square design below we brought the quick-jump interface right at the top of the page so instead of doing work to get where you want, not just click the icon and be taken right there. In-fact we rolled out a click through to nearly every aspect of the website so you can get info and quickly get to the place you want to be.

 

PowerFlow to Experia

 

The next big thing that V20 introduces is major improvements to web technology. Gone are the days of PowerFlow, Meet the all new era of Experia.

 

Experia Is arguably the biggest behind the scenes tech that we have ever brought, from user choice graphical improvements all the way to quick button navigation Experia makes the web that you know even easier. Built from the ashes of PowerFlow we made Experia faster, easier and much more human to use. Allowing for a fluent design that is consistent across the board Experia is a new era for the design language that we use and the engine that powers what you see visually.

 

From surface level down below. Introducing users to the mighty new Hyper Technologies!

 

Introducing HyperEX and HyperConnect. The 2 rad new technologies that power everything you love from behind the scenes. Over the older system that we used, HyperEX and Connect massively boost speed and allow for the best internal web structure yet.

 

Welcome HyperEX, The near surface level tech that makes Inter-Site possible. HyperEX is a new tech that we have built that integrates external HTML files to bring users a all in one experience at the same speed and reliability as the main website you are on. Used across the board HyperEX massively increases speeds for web games and makes everything we do 3x faster than using a dedicated L-NET Server.

 

HyperConnect, Light speed connectivity and high integrative tech! Powering server to server connections the all new HyperConnect is a new tech that we developed to make connecting through other systems possible. Want to access and use news on say the old website? Meet HyperConnect, Want to chat through Solaris chat to a server that we run? Meet HyperConnect. Powering the web that you know and connecting it all together.

 

Boom bash its the sound of a new update system

 

Wow, This is a awesome system you have got here. V20 introduces a new way for us to update. With a all new dedicated development server users are able to test the upcoming releases as they are being developed with the all new build numbers that we have. Although using http and not https the development server that we introduced with V20 means that users are able to see what is coming up and help with the development of the next version. Because sharing is caring.

 

And hey its also much simpler and takes less confusing experimental wackyness!

 

News In Newsroom a better service

 

God remember CNEWS, That is over now. We brought CNEWS down and introduced the all new Newsroom. Using a more generalised press release style of news we re-designed the old CNEWS to work better for slow news like what we push, With articles slowly written and few submitted we needed a way to have a simple to maintain and easy to use new News system and this is where Newsroom comes in. Using links rather than needing graphics every time we have allowed the slowdown and easier writing and development of all news stories while also introducing a slower style.

 

No more needing to write alot to make everything feel alive.

 

Oh and we can also push press statments now rather than having to write articles on everything. Such a more free style.

 

There is just too much to dive into

 

With the sheer level of content that was added into V20 we literally cannot cover it all in detail, there is that much. So why not take what we told you and run around the website, discover some of the featured we did not talk much about such as segmented windows, InfoBox and so so much more.

 

It is all here now in the latest and greatest Cherry Universe V20. Celestial Sweep.
